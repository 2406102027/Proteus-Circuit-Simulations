# Proteus-Circuit-Simulations
This repository contains circuit simulations designed using Proteus.
## Current Project:
### RC TimeConstant Analysis
- Designed and simulated an RC charging and discharging circuit
- Verified time constant(τ = RC) through transient analysis
- Observed exponential charge and discharge behavior using a pulse input.

### Parameters:
- R = 1k- Ω
- C = 1000µF
- τ = RC = 1 second

Software: Proteus 8
Analysis Type: Transient Simulation
