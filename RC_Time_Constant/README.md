# RC Time Constant Analaysis

## Objective
To analyze the charging and discharging behavior of an RC circuit using simulation.

## Circuit Parameters
- R = 1kΩ
- C = 1000µF
- τ = RC = 1 second

## Method
A VPULSE source was used to apply a step input to the circuit.
The capacitor voltage was measured using an oscilloscope.

## Results
The waveform followed an exponential behavior.
The capacitor reached approximately 63% of its final voltage at 1 second.

## Conclusion
The capacitor voltage showed exponential charging and discharging behavior when a pulse input was applied.
The simulation results matched theoretical calculations.
